CHAPTER 01

IDENTITY FILE

私について

なぜ僕がこの思考法を伝えるのか。 まずは、その背景となる僕自身の経験からお話しします。

01 幼少期から感じていた違和感

皆さんこんにちは。著者のアーロンです。 まず初めに私の紹介をしようと思います。

2005年1月13日に愛知県蒲郡市という町で生まれました。 幼少期から、周りとは馴染めないことが多くありました。

幼稚園の頃、先生が「紙芝居を読むからみんな集まって」と言うと、 クラスの全員が一斉に走って集まりました。 しかし僕は一人だけ行かず、窓から車を眺めていました。

当時のことは詳しく覚えていませんが、 昔から人と同じ行動をすることが極端に苦手でした。

02自信があったわけではない

そんな僕も、昔から揺るがない自信を持っていたわけではありません。 学校の勉強は得意ではなく、テストの点数で評価される環境では、 自分に対して前向きな評価を持ちにくい時期もありました。

中学・高校時代は、自分の理想と実際の自分が 最も離れていた時期だったと思います。

03SNSで受けた1万件以上の批判

大学生になってからSNSで発信を続ける中で、 僕は合計1万件を超えるアンチコメントを経験しました。 一つの投稿に数百件の批判が届くこともあります。

それでも発信を続ける中で、 周囲の反応と自分自身の価値を切り離して考える方法を身につけました。 その考え方を、僕は「アーロンマインド」と呼んでいます。

CORE MESSAGE

この教材は、最初から強かった人間の話ではありません。 周囲の評価に揺れていた人間が、 自分の判断基準を取り戻していった過程です。

CHAPTER 02

EXTERNAL AXIS

日本人の98％は他人軸

自分で決めているように見えて、 実際には周囲の評価を基準にしていないでしょうか。

01正解を他人に決めてもらう習慣

学校では、決められた答えを正確に出すことが評価されます。 周囲と違う答えを出せば、 間違いとして扱われることも少なくありません。

その環境が長く続くと、 「自分はどう考えるか」より先に 「周囲からどう見られるか」を考えるようになります。

02集団に合わせることが安全になる

人と同じ進路を選び、人と同じ働き方をし、 人と同じタイミングで安心する。 その方が批判されにくく、失敗しても責任を感じにくいからです。

しかし、周囲に合わせることを続けていると、 自分が本当に何を望んでいるのかが 少しずつ分からなくなっていきます。

03他人軸なのに承認欲求は消えない

周囲に合わせて生きても、 「評価されたい」「認められたい」という感情は消えません。 むしろ、自分で自分を評価できない分だけ、 他人からの承認を必要とするようになります。

承認欲求を完全になくす必要はありません。 ただし、他人からの評価を 自分の価値の最終決定に使わないことが重要です。

CORE MESSAGE

他人軸とは、他人を大切にすることではありません。 自分の人生の決定権を、無意識に他人へ渡している状態です。

CHAPTER 03

MENTAL DEFENCE

アンチが来ても安定しているワケ

アンチコメントが来ることは、 本当に悪いことなのでしょうか。 僕は発信を続ける中で、 批判を失敗ではなく現象として見るようになりました。

01投稿者と視聴者は同じ立場ではない

投稿者は、自分の顔や考えを公開し、 批判される可能性を引き受けながら発信しています。 一方で視聴者は、安全な場所から感想を書くことができます。

その立場の違いを理解すると、 一つ一つのコメントを対等な評価として 受け止める必要がないと分かります。

02批判を評価ではなくデータとして見る

コメントが増えるということは、 それだけ多くの人の感情を動かしたということです。 反応が強い部分には、発信を伸ばすヒントがあります。

感情的に反応する前に、 「なぜこの表現が反応を生んだのか」と観察することで、 批判を改善材料へ変えることができます。

03アンチコメントも人生の作品になる

今は不快に見えるコメントも、 数年後には自分がどれだけ注目されたかを示す記録になります。 苦しい出来事も、時間がたてば物語の一部になります。

「この出来事は、将来どのような意味を持つか」と考えると、 今この瞬間の感情だけに支配されにくくなります。

CORE MESSAGE

批判されないことを目指すのではありません。 批判されても、自分の進む方向を変えない状態を作るのです。

CHAPTER 04

SELF AXIS BUILD

超自分軸の作り方

情報を増やす前に、他人の評価が入ってくる入口を減らします。

01そもそもコメントを見ない

見る必要のない評価まで、自分から取りに行く必要はありません。 重要な改善点を確認した後は、 同じような批判を何度も読み返さないことです。

精神力だけで耐えるのではなく、 自分を不安定にする情報から物理的に距離を取ります。

02SNSを見ない時間を作る

SNSでは、他人の成功や評価が絶えず目に入ります。 それを見続けていると、 自分の人生より他人の人生について考える時間が増えていきます。

発信する時間と、見る時間を分けるだけでも、 自分の思考を保ちやすくなります。

03自分の評価は自分でする

今日の行動が良かったかどうかを、 いいねや再生数だけで判断しないことです。 自分で決めた基準を守れたかを確認します。

他人の評価は参考資料です。 最終的な判定を下すのは、自分でなければなりません。

04自分だけの自己哲学を作る

自分が何を大切にし、何を許せず、 どのような人間でありたいのかを言葉にします。

判断基準が言語化されていれば、 突然批判されたときにも 感情だけで進路を変更しにくくなります。

CORE MESSAGE

自分軸とは、何でも自分勝手に決めることではありません。 自分の選択に、自分で責任を持てる状態です。

CHAPTER 05

MENTAL TRAINING

アーロンメンタルをあなたにも

周囲の評価に振り回されず、 自分の精神状態を自分で管理するためのトレーニング。

01アンチコメントよりキツい習慣を取り入れる

普段から自分で決めた少し厳しい習慣を続けていると、 SNS上の批判や一時的な嫌な出来事が、 以前ほど大きなものに見えなくなります。

大切なのは、自分を苦しめることではありません。 「自分で決めたことを、自分で守った」という経験を積み重ね、 自分に対する信頼を作ることです。

02他人の炎上を自分事として考える

有名人や発信者が炎上したニュースを見たときに、 単に批判する側へ回るのではなく、 「これが自分だったらどう考えるか」と想像します。

事前にさまざまな状況をシミュレーションしておくことで、 実際に批判を受けたときにも感情だけで反応せず、 一歩引いて状況を見られるようになります。

03物事をプラスに変換する脳を作る

どんな出来事にも、悪い面だけでなく利用できる面があります。 批判を受けたなら、注目を集めた証拠として捉える。 失敗したなら、次に避けるべき方法が分かったと考える。

無理に前向きなふりをするのではなく、 「この出来事を自分の人生にどう利用できるか」 と考える習慣を作ることが重要です。

CORE MESSAGE

強いメンタルとは、何も感じないことではありません。 感情が動いた後に、自分の意思で元の状態へ戻れることです。

CHAPTER 06

THINKING MODEL

アーロン思考法を伝授

判断に迷ったときに使える、 自分の本音を確認するための思考法です。

01常に自分はどうしたいのかで決める

世間的に正しいか、周囲から褒められるかを考える前に、 自分が本当に望んでいる方向を確認します。

他人の意見を無視する必要はありません。 意見を聞いたうえで、最後の決定権だけは自分が持ちます。

02地球上に自分一人でも、それをするか

他人から見られなくても続けたい行動なのかを考えます。 誰にも知られなければやめるのなら、 その行動は承認のためだった可能性があります。

この問いは、自分の本心と見栄を分けるために有効です。

03誰にも褒められなくても続けるか

成果が出るまでの期間は、 誰にも評価されないことがほとんどです。 その期間にも続けられることが、本当に自分が選んだ道です。

04自分の評価は自分でする

一日の終わりに、他人の反応ではなく、 自分が決めた行動を実行できたかで評価します。

自己評価の基準を持つと、 周囲の反応が悪い日でも、自信を失いにくくなります。

CORE MESSAGE

他人の意見を聞くことと、 他人に人生を決めてもらうことは別です。

CHAPTER 07

POSITIVE CONVERSION

アーロンのプラス思考法

無理に前向きなふりをするのではなく、 出来事の使い道を探す思考法です。

01物事を表と裏の両方から見る

注目されれば、批判も増えます。 自由になれば、責任も増えます。 一つの出来事には、ほとんどの場合二つ以上の側面があります。

悪い側面だけに集中すると、 本来得られるはずだった利益まで見えなくなります。

02短時間で気持ちを切り替える

感情を無理に消す必要はありません。 一度落ち込んだとしても、 次に何をするかを早く決めることが重要です。

「今できる一番小さな行動は何か」と考えることで、 感情から行動へ意識を移すことができます。

03失敗を素材へ変換する

失敗したことで、うまくいかない方法が一つ分かりました。 批判されたことで、人が反応する点が一つ分かりました。

出来事自体は変えられなくても、 その出来事を何に使うかは自分で決められます。

04「いや、でも、だって」を思考に使う

否定的な言葉も、使い方によっては 思考を深めるための道具になります。

「失敗した。でも経験は残った」 「批判された。だって目立つ行動をしたからだ」 と、別の見方を意識的に追加します。

CORE MESSAGE

ポジティブとは、良いことだけを見ることではありません。 どんな出来事からも、自分に必要なものを回収することです。

CHAPTER 08

AFTER INSTALLATION

アーロンマインドを手に入れた後の世界

自分の評価を自分で決められるようになると、 世界の見え方そのものが変わります。

01ネガティブや後悔に支配されにくくなる

過去の選択を何度も責め続けるのではなく、 当時の自分が持っていた情報の中で選んだ結果として受け止めます。

後悔を自分への攻撃に使うのではなく、 次の判断材料へ変えられるようになります。

02感情が安定する

他人から褒められた日だけ自信を持ち、 批判された日に自信を失うという上下が小さくなります。

自分の価値を毎日他人に決めてもらわなくなるからです。

03他人を羨ましく感じにくくなる

自分の進む方向が明確になると、 他人の成功を自分の敗北として見る必要がなくなります。

他人と競争するよりも、 昨日の自分より前へ進んだかを重視できるようになります。

04失敗という概念が変わる

期待した結果が出なかったことと、 行動そのものが無駄だったことは同じではありません。

経験やデータが残る限り、 その行動は次の選択へつながっています。

05あなたが主人公の世界になる

他人の人生を見ながら、自分を評価する時間が減ります。 自分の選択、自分の挑戦、自分の未来へ 意識を戻せるようになります。

来年、再来年、10年後まで、 自分がどのように変化していくのかを楽しめる状態です。

FINAL MESSAGE

あなたの人生の主人公は、周囲の評価ではありません。 最終的に人生の意味を決めるのは、あなた自身です。
